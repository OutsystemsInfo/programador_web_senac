# 💡 Aula 05 - Iniciando o Proejto Integrador

## 🎯 Objetivo da Aula de Hoje

Hoje você vai dar o primeiro passo rumo ao seu **Projeto Integrador (PI)**!   
A ideia é pensar em um **projeto pessoal** que será desenvolvido ao longo do curso, incluindo **frontend e backend**.   
Este projeto será o seu portfólio e uma demonstração prática de tudo o que você aprendeu.

---

## 🧠 O Que Você Vai Fazer Hoje?

1. **Pensar em uma ideia de projeto**

   - Pode ser um site, sistema ou aplicativo web.
   - Algo que você gostaria de usar, vender ou mostrar para o mundo!

   - Exemplos:
   
     - Um site de receitas com cadastro de usuários.
     - Um sistema de controle financeiro pessoal.
     - Uma agenda online para clientes de salão de beleza.
     - Um blog pessoal com painel administrativo.

2. **Levantar os requisitos do sistema**
   
   - O que o sistema faz?
   - Quem são os usuários?
   - Quais funcionalidades principais?
   - Que problemas ele resolve?

4. **Criar o protótipo no Figma**
   
   - Estrutura das telas (wireframe).
   - Navegação entre páginas.
   - Layout das páginas principais (não precisa estar 100% bonito ainda).
   - Mostre a ideia visualmente!

---

## 🛠️ Ferramentas Sugeridas

- **Figma:** Para o protótipo visual das telas.
- **Google Docs / Notion / Papel:** Para anotar os requisitos.

---

## 📌 Entregável de Hoje

Ao final da aula, você deve ter:

✅ Uma **ideia definida** de projeto.  
✅ Um **documento com requisitos** básicos.  
✅ Um **protótipo inicial** feito no Figma.

---

## 📅 E o que vem depois?

Nas próximas aulas, você vai começar a:

- Criar as páginas com HTML, CSS e JS.
- Desenvolver a API em backend.
- Conectar tudo e deixar seu projeto funcional!

---

> [!TIP]
> Escolha um projeto que você tenha prazer em desenvolver. Isso faz toda a diferença! 🚀
